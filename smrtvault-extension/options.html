smrtVault — settings

One-time setup so the extension can reach your smrtVault.

Backend URL

The address of your smrtesy server (the value of NEXT_PUBLIC_BACKEND_URL — find it in Vercel → Project → Settings → Environment Variables). It looks like https://your-app.up.railway.app.

Auto-lock

Lock the vault after this many minutes of inactivity.

PIN

Reset your unlock PIN. You'll be asked to create a new one next time you open the popup.
