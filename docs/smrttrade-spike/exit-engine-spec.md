# מפרט מנוע-היציאה (T5.1/T5.2) — כלל-אחר-כלל

**מבנה:** מכונת-מצבים יומית על פוזיציה פתוחה. כל יום, בסדר-קדימויות קבוע,
המנוע מחליט: החזק / הרם-סטופ / מימוש-חלקי / יציאה-מלאה. הכל מ-OHLCV
(+ גלאי-הרמות והמעריכים הקיימים). **אין יציאה דיסקרציונית ואין סטופ-זמן שרירותי.**

## מיקום הסטופ ההתחלתי (ח)
- **ח-2:** מתחת לתמיכה (לונג) עם "אוויר".
- **ח-6:** לא ATR-שלם מהרמה (רחב מדי).
- **ח-7:** מרווח יומי 2-3% מתחת לשפל, שבועי 5%.
- **יישום:** `stop0 = support_low − max(0.025×price, 0.5×ATR)`.

## כללי-יציאה (יא) — הבנה + יישום

1. **יא-1 סטופ-עוקב** — מרימים סטופ מתחת לכל שפל-ציר חדש, **רק אחרי שנר סוגר מעליו**.
   יישום: fractal swing-lows; שפל-עולה מאושר → `stop = max(stop, swing_low − אוויר)`.
2. **יא-2 כיוון בלבד** — סטופ רק עולה (לונג). יישום: `stop = max(stop, candidate)`.
3. **יא-3 ב-2R** — "כבר לא מפסידה" → break-even או חצי. יישום: ביעד-1/2R מוכר 50%
   + מזיז סטופ-שארית ל-break-even.
4. **יא-4 לא מוקדם** — אין BE/הידוק לפני היעד-הראשון (guard).
5. **יא-5 מימוש 50/75% + רץ** — ≥50% בהתנגדות, 75% אם המחזור לא-אישר, שארית 10-20%
   לרוץ. יישום: 50% ביעד; 75% אם `pullback_vol_declining=false`/דיברגנס; 15% רץ בטריילינג.
6. **יא-6 לא לפני 1:2** — שער: מימוש חסום עד `profit ≥ 2R` (=י-1).
7. **יא-7 יציאה מבנית** — שבירת שפל-אחרון = שינוי-מגמה → סוגר הכל.
   יישום: `close < last_confirmed_swing_low → יציאה מלאה`.
8. **יא-8 מומנטום** — נר סוגר מתחת לנמוך-הנר-הקודם → יציאה. יישום: `close[t] < low[t-1]`
   (על החלק האגרסיבי / כטריגר-הידוק-סטופ).
9. **יא-9 לא "סתם"** — רק טריגר מוגדר. **מובנה** (אין יציאה דיסקרציונית).
10. **יא-10 אינוולידציה** — נגע בסטופ → יציאה. יישום: `low ≤ stop → יציאה במחיר-הסטופ`.
11. **יא-11 ספק ליד התנגדות** — קרוב לרמה + רווח → לצמצם. יישום: קרוב לרמה-משמעותית
    (גלאי-הרמות) + ברווח → מפעיל מימוש יא-5.
12. **יא-12 מקס' 15** — שער ברמת-תיק על **כניסות**.
13. **יא-13 בדיקה שבועית** — שפל-שבועי חדש → סטופ ~2% מתחתיו. יישום: יא-1 על נרות שבועיים.
14. **יא-14 סטופ אחד לנכס** — פירמידה בנכס = סטופ משותף.
15. **יא-15 לא לסגור לפני 2R** — אלא דוחות קרובים. שער על מימוש-רצוני; חריג `earnings≤7d` (יב-1).
16. **יא-16 גאפ** — סטופ לא מגן מגאפ. יישום: אם `open<stop → מילוי ב-open` (מדמה סיכון-גאפ).

## סדר-קדימויות יומי (לונג)
גאפ (16) → סטופ קשיח (10) → מבני (7) → מומנטום (8) → מימוש-מדורג (3/5/6/11) →
טריילינג (1/2/13) → שער-דוחות (15). הפוזיציה רצה עד שכלל מוציא — אין סטופ-זמן.

## החלק הדיסקרציוני → פרוקסי מחושב
"לפעמים 75%" + כלל-האצבע של שלמה (פחות סימני-היפוך → פחות אגרסיבי) = ספירת
סימני-היפוך (RSI קניית-יתר · דיברגנס-מחזור · ליד-התנגדות · נר-היפוך): יותר → 75/100%,
פחות → 50% ולתת לרוץ.

## מה נשאר מחוץ (מוגבל)
- זיהוי-תבנית עדין ליעד-מדוד (ד-4/5/8) — כבר חלקית; משתלב עם גלאי-הרמות.
- Vision — לא נדרש למנוע-היציאה (הכל מ-OHLCV).
