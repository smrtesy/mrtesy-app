# אוצר-מילים לתוויות הגולדן-סט (ההחלטה המעשית, לא הסנטימנט)

**הבחנה מרכזית:** התווית מתעדת את ה**החלטה המעשית** של זיו, לא את סנטימנט-המגמה.
"מגמה עולה אך לממש רווחים / לא להצטרף" הוא **לא כניסה** — הוא מעקב/מימוש.
(תיוג אוטומטי מוקדם בלבל "bullish" עם "entry"; זה תוקן ידנית מול הציטוטים.)

| תווית | פירוש | תווית-על להשוואה |
|---|---|---|
| **כניסה** | איתות-קנייה מפורש לעכשיו | כניסה |
| **מעקב** | אין כניסה עכשיו — ממתינים / לא-אטרקטיבי / קרוב-התנגדות | מעקב |
| **מעקב·מימוש** | מגמה עולה אך זיו ממליץ לממש/לא להצטרף (ניואנס-יציאה) | מעקב |
| **מעקב·אזור-כניסה** | כניסה מותנית עתידית (בפריצה/חזרה לרמה מוגדרת) | מעקב |
| **הימנעות** | חולשה / ירידה / שורט | הימנעות |
| **הימנעות·מותנית** | שלילי רק אם רמה נשברת | הימנעות |

**כלל-מדידה:** תיאום תווית-על (כניסה/מעקב/הימנעות) הוא המדד הגס. אך המדד
**המבחין** הוא רק על מקרים שבהם מישהו אמר "כניסה" — שם נמדדת יכולת-הזיהוי
האמיתית (מעקב-מול-מעקב ביום שקט הוא הסכמה זולה).
