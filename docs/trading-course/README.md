# כללי המסחר — קורס Cycles Trading

מסמך העבודה המרכזי של כללי המסחר מהקורס, כפי שחולצו **מכל** חומרי הלימוד: 60 תמלולי שיעורים, 37 קובצי PDF של סיכומים רשמיים, והמחברת האישית של חנוך. נבנה כבסיס לשני שימושים: (1) מערכת שתנתח גרפים של מניות לפי כללי הקורס; (2) אפליקציית שאלות-תשובות אמינה על חומרי הקורס.

## איך המסמך נבנה

1. **חילוץ** — כל תמלול, כל PDF והמחברת נקראו במלואם; כל כלל חולץ עם ציטוט מילולי מדויק מהמקור.
2. **נורמליזציה** — 1,356 רשומות כללים הומרו לפורמט אחיד (קטגוריה, הקשר, משקל, ישימות, ציטוט, מקור).
3. **איחוד** — כלל שמופיע בכמה מקורות (תמלול + PDF + מחברת) אוחד לכלל קנוני אחד עם כל מקורותיו. **התמלול הוא המקור המוסמך**; ה-PDF והמחברת משמשים לאימות. סתירות בין מקורות **לא הוכרעו** — הן מוצגות בשקיפות.

התוצאה: **624 כללים ממוסמרים וממוספרים**, כל אחד עם מזהה קבוע (למשל `ENT-07`) שאפשר להתייחס אליו בשיחה.

## ⚠️ הערת מספור חשובה

**שמות קובצי התמלול אינם תואמים את מספרי השיעורים האמיתיים** (למשל: הקובץ `lesson_19` הוא בפועל שיעור 17, והקובץ `lesson_28` הוא שיעור 25 — וההסטה אינה אחידה). כל מספרי השיעורים במסמכים נלקחו **מתוך התמלול עצמו** ("ברוכים הבאים לשיעור מספר..."). טבלת המיפוי המלאה — ב[קטלוג השיעורים](lessons-catalog.md). הקורס הממוספר: שיעורים 1–37, ובנוסף סדרת כלי גאן (4 שיעורים), בונוסים, מפגשי לייב ונספחים.

## אזהרות אמינות

1. **התמלולים אוטומטיים** ומכילים שיבושים (למשל "נמס" = נמ"ס, "ETR" = ATR). ציטוטים הובאו כלשונם; שיבושים שמשנים משמעות רוכזו בסעיף 95.
2. **ה-PDF חולצו עם היפוך RTL** — התוכן שוחזר, אך ייתכנו שיבושי סדר-מילים נקודתיים.
3. **מקור משני** (קובץ חילוץ ישן שנמצא בדרייב) — כללים שמקורם היחיד בו מסומנים "טעון אימות".
4. כללים שסומנו 🔒/➕ — רק על סמך לשון מפורשת של המרצה; היעדר סימון אינו אומר שהכלל פחות חשוב.
